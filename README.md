# 📈 Stock Price Analysis using Technical Indicators & Dandelion Optimizer

This project focuses on analyzing stock market data using technical indicators and optimizing prediction performance using the Dandelion Optimizer.

## 🚀 Overview
The goal of this project is to enhance stock price prediction by combining technical analysis indicators with an optimization algorithm. It demonstrates how data preprocessing, feature engineering, and optimization techniques can improve model performance.

## 🛠️ Tech Stack
- Python
- Pandas
- NumPy
- Matplotlib
- Scikit-learn

## 📊 Key Features
- Data preprocessing and cleaning of stock market data
- Implementation of technical indicators such as:
  - Simple Moving Average (SMA)
  - Exponential Moving Average (EMA)
  - Relative Strength Index (RSI)
- Feature engineering using financial indicators
- Application of **Dandelion Optimizer** for improving model performance
- Visualization of stock trends and indicators

## 🔗 Workflow (Pipeline)
1. Data Collection (stock dataset)
2. Data Cleaning & Preprocessing
3. Feature Engineering (technical indicators)
4. Model Training
5. Optimization using Dandelion Optimizer
6. Evaluation and Visualization

## 📈 Results
- Improved prediction performance using optimization techniques
- Better understanding of stock trends through indicators
